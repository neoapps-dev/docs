OAuth API Documentation

• First Step: Register App

The first thing to do is registering the app on aSocial.

To do this, Open aSocial > Click on the Profile Picture > Click the menu icon > Settings > Extra > New OAuth App

Write your app name, package name and select your app icon.

after that, a link will appear. that link is your Authentication Token.

Make sure to save it safe.
